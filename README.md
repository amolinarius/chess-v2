# ♟️ Chess

A fully client-side object-oriented Chess game playable in a browser

## How to install

- Download Node.js and npm
- Run `npm i` in any terminal
- Run `npm run dev` for development server or `npm run build` for static server

## Features

- [ ] Basic Movement, Castling, En Passant, Promotion
- [ ] Player Information
- [ ] Bot (Stockfish) Integration
- [ ] Evaluation Bar
- [ ] Moves List
- [ ] Ability to review previous positions (Arrows)
- [ ] Opening Recognition
- [ ] "Open in WikiBooks' Chess Opening Theory" button
- [ ] Win Detection + Popup
- [ ] Annotation (Right-click)
- [ ] Visible Evaluation Bar value
- [ ] Ability to play Black
- [ ] Game Analysis
- [ ] Premoving
- [ ] Draggable Pieces
